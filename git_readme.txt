Github steps

On main PC

Init rep
$ git init

connect to rep
$ git remote add origin https://github.com/varvar6666/moto_stm32f405.git

add files to commit
$ git add --all

make commit
$ git commit -m "NOTE"

push to server first commit
$ git push -u origin master

Create dev branch
$ git branch develop

switch to dev branch
$ git checkout develop

list of branches
$ git branch #
